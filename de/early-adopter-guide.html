Leitfaden fur Erstnutzer

Installieren Sie die App in weniger als 5 Minuten

EARLY-ACCESS-PROGRAMM

Vielen Dank, dass Sie dabei sind! Sie gehoren zu den Ersten, die Wie geht's?! ausprobieren — eine App, die mit KI die Sicherheit alterer Eltern uberwacht. Keine Armbander, keine Kameras, keine taglichen Anrufe. Nur ihr Android-Handy.

Folgen Sie den Schritten unten. Alles dauert etwa 5 Minuten.

1
Treten Sie der Erstnutzer-Gruppe bei

Offnen Sie den Link unten und tippen Sie auf "Gruppe beitreten" (oder "Beitritt beantragen"). Damit wird Ihr Google-Konto registriert, sodass Sie die App herunterladen konnen.

Google-Gruppe beitreten groups.google.com/g/howareu-early-adopters
Verwenden Sie dasselbe Google-Konto, das auf Ihrem Android-Handy aktiv ist. Wenn Sie mehrere Google-Konten haben, wahlen Sie das Konto, das Ihr Handy verwendet.
2
Nehmen Sie die Testeinladung an

Offnen Sie diesen Link. Sie sehen eine Seite von Google Play:

Testeinladung offnen play.google.com/apps/testing/com.howareyou

Tippen Sie auf den grunen Button "Become a tester" (Tester werden):

Tippen Sie auf "Become a tester"

Nach der Annahme andert sich die Seite. Sie sehen "You are a tester" (Sie sind ein Tester) und einen Link zum Herunterladen der App:

Sie sind jetzt Tester — tippen Sie auf "download it on Google Play"

Fertig! Sie sind jetzt registriert. Fahren Sie mit Schritt 3 fort, um die App herunterzuladen.
3
Laden Sie die App herunter

Wahlen Sie, was fur Sie einfacher ist:

📱

Auf dem Handy

Offnen Sie diesen Link direkt auf Ihrem Android-Handy und tippen Sie auf "Installieren"

Google Play offnen
🌐

Im Web

Besuchen Sie unsere Website von jedem Gerat und tippen Sie auf "Herunterladen"

howareu.app besuchen
Wenn Google Play "App nicht verfugbar" anzeigt — gehen Sie zuruck zu Schritt 1 und prufen Sie, ob Sie der Gruppe beigetreten sind und die Einladung mit demselben Google-Konto angenommen haben, das auf Ihrem Handy aktiv ist.
4
Losen Sie Ihren Promo-Code ein

Als Erstnutzer erhalten Sie die App vollig kostenlos. Ihr personlicher Promo-Code wird in der Google-Gruppe geteilt.

Ihr Promo-Code
XXXX-XXXX-XXXX
Prufen Sie die Google-Gruppe fur Ihren personlichen Code

So losen Sie ihn ein:

  • Offnen Sie die App und gehen Sie zum Abo-Bildschirm
  • Tippen Sie auf "Abonnieren" — dies offnet die Google Play Kasse
  • Sehen Sie sich die Liste der Zahlungsmethoden an
  • Scrollen Sie zum Ende der Liste und tippen Sie auf "Code einlosen"
  • Geben Sie Ihren Promo-Code ein und tippen Sie auf "Einlosen"
Das war's! Die App ist jetzt vollstandig freigeschaltet — ohne Kosten fur Sie. Keine Kreditkarte erforderlich.
Jeder Code funktioniert nur einmal und ist mit einem Google-Konto verknupft. Bitte teilen Sie Ihren Code nicht mit anderen.
5
Richten Sie die App ein

Offnen Sie die App und folgen Sie den Anweisungen auf dem Bildschirm. Die Einrichtung dauert etwa 3 Minuten:

  • Berechtigungen erlauben — die App benotigt Standort- und Aktivitatsberechtigungen fur die Sicherheitsuberwachung
  • Akkuoptimierung deaktivieren — damit die App rund um die Uhr im Hintergrund laufen kann (die App fuhrt Sie durch den Vorgang)
  • Familien-E-Mail hinzufugen — diese Person wird benachrichtigt, wenn etwas Ungewohnliches erkannt wird
Nach der Einrichtung: Die App startet eine 7-tagige Lernphase, in der sie die taglichen Gewohnheiten leise beobachtet. Danach beginnt die Uberwachung automatisch. Sie mussen nichts weiter tun — die App ist so konzipiert, dass sie nach der Installation von alleine lauft.
